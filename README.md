# Passive

Identity Management for ACM@UIC written in C# and Typescript with .NET Core and Angular.

## Getting Started

- `dotnet build`
- `dotnet run`

# Configuration

If not running as user with appropriate privleges in AD, the following environment variables are required.

- `PASSIVE_AD_HOST`
- `PASSIVE_AD_USER`
- `PASSIVE_AD_PASSWORD`
